üì¶ Installation

The easiest way to get started is by using the CDN link. No build step required.

<script src="https://cdn.jsdelivr.net/npm/hubdox-chat-sdk@1"></script>

Alternatively, you can install via NPM:

npm install hubdox-chat-sdk

üõ†Ô∏è Basic Usage

Instantiate the chatbot with your required credentials. The chatbot will appear automatically.

const chat = new ChatBot({
  baseUrl: 'https://your-api.com',
  apiKey: 'your-api-key',
  tenant: 'your-tenant'
});

‚öôÔ∏è Configuration

The SDK is configured by passing an object to the constructor. Here are the main properties:

Required Options

ParameterTypeDescription
baseUrlstringBase URL of your chat API
apiKeystringAPI Key for authentication
tenantstringIdentifier for your organization

Optional `options` Object

ParameterTypeDefaultDescription
registerbooleanfalseEnables conversational user registration
testModebooleanfalseEnables test mode with mock responses
cachebooleantrueEnables session and message caching

Optional `custom` Object

Used for deep UI customization.

ParameterTypeDescription
primaryColorstringMain accent color for the widget
headerBgColorstringBackground color of the chat header
botNamestringDisplay name for the bot in the header
positionobjectSets the position (e.g., { bottom: '24px', left: '24px' })
iconButtonstringURL for a custom floating button icon
sendButtonTextstringCustom text for the send message button

‚ú® 30 Live Examples

Scroll through these examples to see the chatbot update in real-time on the right. Each example demonstrates different configurations and features.

1. Default Minimal

The chatbot with its default settings. Clean and simple.

new ChatBot({
  baseUrl: 'https://api.example.com',
  apiKey: 'your-api-key',
  tenant: 'your-tenant',
  options: {
    testMode: true
  }
});

2. Registration Screen

Shows the new registration screen where the bot greets and asks for the user's name.

new ChatBot({
  baseUrl: 'https://api.example.com',
  apiKey: 'your-api-key',
  tenant: 'your-tenant',
  options: {
    register: true,
    testMode: true
  },
  user: {
    name: "Usuario",
    email: "usuario@ejemplo.com"
  }
});

3. Custom Colors
