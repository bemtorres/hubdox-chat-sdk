üì¶ Installation

The easiest way to get started is by using the CDN link. No build step required.

<script src="https://cdn.jsdelivr.net/npm/hubdox-chat-sdk@1"></script>

Alternatively, you can install via NPM:

npm install hubdox-chat-sdk

üõ†Ô∏è Basic Usage

Instantiate the chatbot with your required credentials. The chatbot will appear automatically.

const chat = new ChatBot({
  baseUrl: 'https://your-api.com',
  apiKey: 'your-api-key',
  tenant: 'your-tenant'
});

‚öôÔ∏è Configuration

The SDK is configured by passing an object to the constructor. Here are the main properties:

Required Options

ParameterTypeDescription
baseUrlstringBase URL of your chat API
apiKeystringAPI Key for authentication
tenantstringIdentifier for your organization

Optional `options` Object

ParameterTypeDefaultDescription
registerbooleanfalseEnables conversational user registration
testModebooleanfalseEnables test mode with mock responses
cachebooleantrueEnables session and message caching

Optional `custom` Object

Used for deep UI customization.

ParameterTypeDescription
primaryColorstringMain accent color for the widget
headerBgColorstringBackground color of the chat header
botNamestringDisplay name for the bot in the header
positionobjectSets the position (e.g., { bottom: '24px', left: '24px' })
iconButtonstringURL for a custom floating button icon
sendButtonTextstringCustom text for the send message button

‚ú® Live Examples

Scroll through these examples to see the chatbot update in real-time on the right.

Default Minimal

The chatbot with its default settings. Clean and simple.

new ChatBot({
  baseUrl: 'https://api.example.com',
  apiKey: 'your-api-key',
  tenant: 'your-tenant',
  options: {
    testMode: true
  }
});

Custom Colors

Change the primary color, header background, and text color to match your brand.

new ChatBot({
  // ...
  custom: {
    primaryColor: '#ff6b35',
    headerBgColor: '#2c3e50',
    headerTextColor: '#ecf0f1',
    botName: 'Orange Bot'
  }
});

Custom Position (Bottom Left)

Move the floating button to a different corner of the screen.

new ChatBot({
  // ...
  custom: {
    position: {
      bottom: '24px',
      left: '24px',
      right: 'auto'
    }
  }
});

Custom Position (Top Right)

Another example of custom floating button placement.

new ChatBot({
  // ...
  custom: {
    position: {
      top: '24px',
      right: '24px',
      bottom: 'auto'
    }
  }
});

Conversational Registration

Enable the built-in conversational registration flow to collect user information.

new ChatBot({
  // ...
  options: {
    register: true,
    testMode: true
  }
});

Custom Floating Button Icon

Use your own image for the floating chat button.

new ChatBot({
  // ...
  custom: {
    iconButton: 'https://www.svgrepo.com/show/508699/chat-bubble.svg'
  }
});

Custom Send Button Text

Change the text displayed on the message send button.

new ChatBot({
  // ...
  custom: {
    sendButtonText: 'Send Now!'
  }
});

No Cache

Disable the chat history caching. Each refresh will start a new session.

new ChatBot({
  // ...
  options: {
    testMode: true,
    cache: false
  }
});
