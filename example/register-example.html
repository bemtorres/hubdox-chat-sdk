🚀 Sistema de Registro de Usuarios

¿Cómo funciona?

Este ejemplo demuestra el sistema de registro obligatorio. El usuario debe proporcionar su nombre antes de poder chatear con el bot.

Flujo de Registro:

  1. Inicio: El bot se conecta y obtiene la configuración desde la API
  2. Solicitud de nombre: El bot pregunta al usuario su nombre
  3. Validación: Se valida que el nombre no esté vacío
  4. Registro: Se guarda el nombre en el caché local
  5. Activación: El chat se habilita y el usuario puede empezar a conversar

💬 Haz clic en el botón flotante para abrir el chat y probar el sistema de registro

El chat aparecerá en la esquina inferior derecha

Información del Estado:

Puedes verificar el estado del registro abriendo la consola del navegador y ejecutando:

chat.getRegistrationStatus()

O verificar el caché con:

chat.getCacheStatus()
